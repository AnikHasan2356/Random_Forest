# Decision Tree & Random Forest Classification using Scikit-learn

This repository demonstrates the implementation of Decision Tree and Random Forest classifiers for supervised machine learning using Scikit-learn. It covers model training, hyperparameter tuning with GridSearchCV, model evaluation, and performance comparison on the Breast Cancer dataset.

## Topics Covered

- Decision Tree Classification
- Understanding tree depth (`max_depth`)
- Preventing overfitting and underfitting
- Hyperparameter tuning using GridSearchCV
- Random Forest (Ensemble Learning)
- Optimizing the number of trees (`n_estimators`)
- Feature selection strategies (`max_features`)
- Model evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1-Score
  - Confusion Matrix
  - Classification Report
- Comparing Decision Tree and Random Forest performance

## Dataset

- Breast Cancer Wisconsin Dataset
- Loaded directly from `sklearn.datasets`

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Purpose

The purpose of this project is to understand tree-based machine learning algorithms and ensemble learning techniques. It demonstrates how hyperparameter tuning with GridSearchCV can improve model performance and compares the predictive capabilities of Decision Trees and Random Forests.
